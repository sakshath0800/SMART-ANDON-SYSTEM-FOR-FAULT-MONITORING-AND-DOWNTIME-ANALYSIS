# SMART-ANDON-SYSTEM-FOR-FAULT-MONITORING-AND-DOWNTIME-ANALYSIS
Develop a Smart Andon System for industrial fault monitoring capable of detecting faults in real time, classifying faults, automatically monitoring downtime, recording fault occurrences, maintaining historical fault records and generating analytical reports.


Problem Statement:

Frequent machine faults and delayed issue identification lead to increased downtime and reduced production efficiency in industrial environments.


Proposed Solution: 

Develop a Smart Andon System for industrial fault monitoring capable of detecting faults in real time, classifying faults, automatically monitoring downtime, recording fault occurrences, maintaining historical fault records and generating analytical reports.


Methodology:

1. Industrial sensors continuously monitor machine conditions. 
2. Tool Failure is detected using an inductive proximity sensor. 
3. Material Shortage is detected using a photoelectric sensor. 
4. Overload relay monitors motor current. 
5. PLC diagnostic logic checks sensor health. 
6. PLC receives all fault signals. 
7. PLC identifies and classifies the fault. 
8. Critical faults stop the machine and start downtime monitoring. 
9. Sensor faults generate warnings while allowing operation. 
10. HMI displays machine status and fault information. 
11. SCADA records faults and downtime. 
12. Historical database store records. 
13. Maintenance reports are generated automatically.


Expected Outputs:

• Real-time fault detection
• Automatic fault classification 
• Machine protection 
• Automatic downtime monitoring 
• Historical fault records 
• Automated maintenance reports 
• Improved maintenance planning 
• Enhanced production reliability.
